# Changelog

All notable changes to HazardousWorld will be documented in this file.

## [1.5.0] - 2026-01-07

### Changed
- All items now appear in unified "Modded" build menu tab
- Updated dependency on TechtonicaFramework 1.2.0
- Environmental hazard system improvements

### Fixed
- Build menu organization for cleaner UI

## [1.0.0] - 2025-01-05

### Added
- Initial release
- **Hazmat Suit** - Protective equipment against toxic environments
- **Radiation Shield** - Protection from radiation damage
- **Antidote** - Cures poison and toxic effects
- Integration with TechtonicaFramework for environmental hazard system
