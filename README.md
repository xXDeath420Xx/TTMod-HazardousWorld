# Hazardous World

Adds environmental hazards and protective equipment to Techtonica.

## Features

### Protective Equipment
- **Hazmat Suit** - Protects against toxic environments
- **Radiation Shield** - Protects against radiation damage
- **Antidote** - Cures poison and toxic effects

## Requirements

- BepInEx 5.4.21+
- EquinoxsModUtils 6.1.3+
- EMUAdditions 2.0.0+
- TechtonicaFramework 1.0.0+

## Installation

Install via r2modman or manually place the DLL in your BepInEx/plugins folder.
